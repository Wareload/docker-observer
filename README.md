## How do I run this?

There are multiple ways to run this locally.

### Docker Compose

You need to have docker installed.

Start the application:
``` bash
docker compose up -d
```

Stop the application:
``` bash
docker compose down
```

### Plain

You need to have node v20 installed.

Start the application:
``` bash
npm run dev
```
